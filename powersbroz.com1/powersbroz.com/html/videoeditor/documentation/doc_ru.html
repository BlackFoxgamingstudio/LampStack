Web Video Editor

JS / PHP script - GUI for FFmpeg (and Mencoder)

  • version: 1.5
  • created: 09/29/2014
  • latest update: 14/02/2015
  • by: Andycoder
  • email: andycoderw@gmail.com

Описание

JS / PHP скрипт онлайн видеоредактора, GUI для FFmpeg (или Mencoder). Позволяет разрезать видео на куски и соединять их. Конвертация видео в форматы: FLV, MP4, WEBM, OGV. Возможность быстрого разрезания видео на фрагменты без переконвертации.

Возможности и особенности

  • Мгновенное разделение видео на куски без переконвертации.
  • Возможность склеивания кусков видео.
  • Для работы с видео используются серверные приложения: FFmpeg, Mencoder (не обязательно), MP4Box / gpac (не обязательно), MKVMerge (не обязательно).
  • Конвертация видео в форматы: FLV, MP4, WEBM, OGV.
  • Возможность редактирования аудио MP3 файлов.
  • Показ визуализации аудио-волны при редактировании MP3.
  • Показ прогресса конвертации в процентах.
  • Три HTML5 плеера на выбор.
  • Загрузка видео 3-мя способами: файл, по ссылке, с Youtube.
  • Список пользователей можно задать в конфигурационном файле или можно настроить авторизацию из таблицы базы данных MySQL.
  • Простая конфигурация (в файле).
  • Мультиязычность (сделаны переводы на Английском и Русском языках).
  • Приятный интерфейс и адаптивная верстка (Bootstrap 3.2).
  • Открытый чистый исходный код.

Требования к системе

Обязательные:

  • Linux или Windows сервер
  • PHP 5.3+
  • Веб-сервер Apache или Nginx
  • FFmpeg 1.2+

Не обязательные:

  • Mencoder
  • MP4Box (gpac)
  • MKVMerge

Установка

  1. Установить серверные приложения (инструкция ниже).
  2. Загрузить содержимое архива videoeditor на сервер (например в папку videoeditor/).
  3. Установить права 777 на папки: input/, output/, tmp/.
  4. Открыть в браузере адрес папки, в которую загрузили файлы. Например: http://[адрес сайта]/videoeditor/.
  5. Открыть в текстовом редакторе файл config.php и настроить конфигурацию (в файле есть комментарии).
  6. Готово.
Создание MySQL таблицы для авторизации пользователей

В config.php можно задать список пользователей ( $config['users'] ). Также авторизация пользователей может происходить по данным из базы данных MySQL.

  1. Создайте таблицу в базе занных MySQL.
  2. Заполните данные конфигурации для подключения к этой базе занных в config.php ( $config['db'] ).
  3. Если таблица пользователей не создана, создайте её, используя пример структуры из файла db_users_example.sql.
    Для этого, например, можно использовать phpMyAdmin. Откройте таблицу и затем перейдите на страницу "SQL". Вставьте в поле запрос из файла db_users_example.sql. Нажмите "OK".
  4. Затем в таблице нужно создать пользователей. Типы пользователей могут быть: "admin" и "user". Пароль нужно шифровать по алгоритму MD5.
  5. Готово.

Установка серверных приложений

Для работы видеоредактора необходим только FFmpeg, остальные приложения по желанию, если по каким либо причинам FFmpeg не устроит.
Например, FFmpeg точно разрезает видео в установленных местах, но в начале видеофрагмента будет остановка видеопотока. Остальные режут видео лучше, но не точно в установленных местах (по ключевым кадрам).

Установка FFmpeg на Windows

Скачать FFmpeg для Windows можно здесть: http://ffmpeg.org/download.html#build-windows.

Распаковать архив и загрузить содержимое на диск. Например, адрес должен получиться такой:
"C:\ffmpeg\bin\ffmpeg.exe"

В config.php нужно указать этот адрес в параметре $config['ffmpeg_path'].

Установка FFmpeg на Linux

Сайт разработчика: http://ffmpeg.org/

Устанавливать FFmpeg рекомендуется по этой инструкции: https://trac.ffmpeg.org/wiki/CompilationGuide/Ubuntu
Нужно установить все дополнительные приложения, которые перечислены в данной инструкции.

Внимание!
Настоятельно рекомендуется устанавливать все приложения в папки /usr/bin/ и /usr/ffmpeg_build/.
Для этого в инструкции нужно заменить все "$HOME" на "/usr". Для устновки откройте командную строку (терминад).
Пример: 

mkdir ~/ffmpeg_sources
cd ~/ffmpeg_sources
wget http://ffmpeg.org/releases/ffmpeg-snapshot.tar.bz2
tar xjvf ffmpeg-snapshot.tar.bz2
cd ffmpeg
PATH="$PATH:/usr/bin" PKG_CONFIG_PATH="/usr/ffmpeg_build/lib/pkgconfig" ./configure \
  --prefix="/usr/ffmpeg_build" \
  --extra-cflags="-I/usr/ffmpeg_build/include" \
  --extra-ldflags="-L/usr/ffmpeg_build/lib" \
  --bindir="/usr/bin" \
  --enable-gpl \
  --enable-libass \
  --enable-libfdk-aac \
  --enable-libfreetype \
  --enable-libmp3lame \
  --enable-libopus \
  --enable-libtheora \
  --enable-libvorbis \
  --enable-libvpx \
  --enable-libx264 \
  --enable-nonfree \
  --enable-x11grab
PATH="$PATH:/usr/bin" make
sudo make install
make distclean
hash -r

Установка Mencoder (не обязательно) - Linux

Сайт разработчика: http://www.mplayerhq.hu/

Для Ubuntu (Debian) Mencoder можно установить так:

sudo apt-get install mencoder

Установка MP4Box / gpac (не обязательно) - Linux

Сайт разработчика: http://gpac.wp.mines-telecom.fr/mp4box/

Для Ubuntu (Debian) MP4Box можно установить так:

sudo apt-get install gpac

Установка MKVMerge (не обязательно) - Linux

Сайт разработчика: https://www.bunkus.org/videotools/mkvtoolnix/

Для Ubuntu (Debian) MKVMerge можно установить так:

sudo apt-get install mkvtoolnix

Для других систем инструкции по установке можно найти на сайтах разработчиков.
